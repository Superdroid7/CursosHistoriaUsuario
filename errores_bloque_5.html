Registro de Errores y Soluciones: Bloque 5

Documentación de las dificultades técnicas enfrentadas durante el desarrollo de la HU-013: Carga de Imágenes.

Dificultad 1: Error 401 Unauthorized al visualizar imágenes

Seguridad / Autenticación

El problema: Tras subir exitosamente la imagen al servidor, se generó la URL http://localhost:8080/uploads/foto.png. Sin embargo, al intentar abrir este enlace desde el navegador o la landing page pública, el sistema lanzaba un error de acceso denegado (401 Unauthorized).

La solución: El sistema de seguridad de Spring Security (creado en el bloque 2) por defecto bloquea cualquier petición que no tenga un token JWT, incluyendo la carpeta estática. Se tuvo que modificar el archivo SecurityConfig.java para añadir la ruta /uploads/** a la lista de URLs permitidas públicamente (permitAll()).

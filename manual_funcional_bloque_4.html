Manual Funcional: Gestionando Categorías

¡El sistema ya permite administrar categorías de verdad, completando así la posibilidad de crear cursos reales!

¿Qué hace el sistema ahora?

  • Crear y Editar Categorías: El administrador puede definir las categorías base de la plataforma (ej. Desarrollo, Marketing, Finanzas).
  • Borrado Físico: Si eliminas una categoría, esta desaparecerá por completo para siempre (Hard Delete).
  • Seguro de Vida (Anticorrupción): El sistema te impedirá borrar una categoría si tienes cursos publicados dentro de ella. Esto evita que tu base de datos se rompa o que tus cursos queden flotando en el limbo.

Guía de Pruebas en Swagger

Paso 1: Autorizar y Crear Categoría
1. Consigue tu token en /api/auth/login y ponlo en el botón Authorize de arriba.
2. Ve a POST /api/admin/categories y crea una categoría enviando un JSON como este:
{ "name": "Programación" }
Anota el ID que te devuelve el sistema (por ejemplo, el ID será 1).
Paso 2: Crear un Curso (Por fin)
Ve a POST /api/admin/courses (del bloque 3 que no pudimos probar antes) y ahora sí, crea tu curso asignándole el ID de la categoría que acabas de crear:

{
  "title": "Curso Java",
  "description": "Aprende backend",
  "imageUrl": "link.png",
  "courseUrl": "link.com",
  "categoryId": 1
}
Paso 3: Probar el Seguro de Vida
Ahora intenta borrar la Categoría 1 usando DELETE /api/admin/categories/1. Verás que Swagger te devuelve un error en rojo con el código 409 Conflict. ¡El sistema de seguridad funciona y te está impidiendo destruir la categoría porque "Curso Java" sigue dentro de ella!

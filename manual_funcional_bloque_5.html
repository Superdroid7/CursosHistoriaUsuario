Manual Funcional: Subida de Imágenes

¡El sistema ya permite subir archivos de imagen reales y asociarlos a los cursos! Adiós a los links muertos.

¿Qué hace el sistema ahora?

  • Carga de Archivos Reales: Ahora puedes adjuntar una foto desde tu computadora al crear un curso.
  • Filtro de Seguridad: El sistema rechazará automáticamente cualquier archivo que no sea JPG o PNG (por ejemplo, rechazará PDFs o archivos de virus .exe).

Guía de Pruebas en Swagger

Paso 1: El Curso Base
Asegúrate de haber creado al menos un Curso en el bloque anterior (digamos, el curso con ID 1).
Paso 2: Subir la Imagen
1. Inicia sesión en Swagger (con el candadito `Authorize`).
2. Busca el nuevo endpoint POST /api/admin/courses/{id}/image.
3. En id escribe 1 (el ID de tu curso).
4. Verás un botón que dice Choose File. Dale clic y selecciona cualquier foto JPG o PNG de tu computadora.
5. Presiona Execute.
Paso 3: Verificar la Subida
Si todo salió bien, Swagger te devolverá los datos del curso, pero notarás que el atributo imageUrl ahora dice algo como http://localhost:8080/uploads/2f1b4a-tu_foto.png.

Si copias ese link y lo pegas en una nueva pestaña del navegador, ¡podrás ver la foto! (Ese es el mismo link que la página web de los visitantes usará para mostrarla en pantalla).

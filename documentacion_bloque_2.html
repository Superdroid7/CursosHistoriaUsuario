Reporte de Misión: Bloque 2 - Sistema de Seguridad (JWT)

Agente, la segunda fase está completa. Hemos implementado un sistema de autenticación robusto para separar a los civiles de los operativos autorizados.

1. Orden de Creación: Protegiendo la base

  • Dependencias (Armería): Añadimos jjwt a Gradle para poder fabricar y verificar credenciales digitales (Tokens).
  • Modelo Admin (El VIP): Creamos la entidad Admin a nivel de Dominio e Infraestructura.
  • Seguridad (El Muro de Demacia): Creamos los filtros y configuraciones (SecurityConfig, JwtAuthenticationFilter, JwtService). Aquí le dijimos al sistema: "Deja la zona /api/public abierta, pero pon guardias en el resto".
  • Casos de Uso y Controladores (El Control de Acceso): Implementamos AuthUseCase y AuthController para que los usuarios puedan presentar sus credenciales y recibir su pase de acceso (Token).
  • Data Initializer (El Fundador): Creamos un script que, al iniciar, inserta un admin si la base de datos está vacía.

2. Bloques de Código Clave

Analogía de League of Legends: El JwtAuthenticationFilter es como un Centinela de Control (Pink Ward). Se coloca justo en la entrada de tus rutas y revisa a cada visitante invisible (las peticiones HTTP). Si el visitante no lleva el emblema de tu equipo (un Token JWT válido), el centinela lo rechaza y no le deja interactuar con la torre (el backend).

El SecurityConfig es el estratega: desactiva el CSRF (ya que no usamos sesiones sino tokens), establece las reglas de qué rutas son públicas, y posiciona a nuestro filtro Centinela en la cadena de mando.

3. Anotaciones Especiales

  • @EnableWebSecurity: Activa el radar de seguridad principal de Spring.
  • @Bean: Le dice a Spring: "Toma este objeto (como el codificador de contraseñas BCrypt) y guárdalo en tu inventario principal, lo necesitaremos luego".
  • @Value("${jwt.secret}"): Extrae la llave maestra criptográfica de tu archivo application.yml.
  • @CommandLineRunner: Convierte una clase en un ejecutable que corre justo después de que la aplicación termina de arrancar, ideal para inyectar al administrador por defecto.

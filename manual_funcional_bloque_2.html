Manual Funcional: Autenticación de Administrador

Pruebas en Swagger

[Satisface HU-004]
Paso 1: Iniciar Sesión (Login)
1. Ve a /api/auth/login.
2. Ingresa `admin@ias.com` y `admin`.
3. Copia el token JWT generado.
Resultado: Estás autenticado.
[Satisface HU-005]
Paso 2: Cerrar Sesión (Logout y Blacklist)
1. Ve a /api/auth/logout.
2. Presiona Execute teniendo tu token pegado en el candado verde de Swagger.
Resultado: El servidor responderá con un código HTTP 204 (No Content). El token que estabas usando acaba de ser metido en la tabla de revoked_tokens.
3. Intenta usar el MISMO token para crear una categoría. El sistema te arrojará 401 Unauthorized confirmando que el token fue invalidado y no sirve más.

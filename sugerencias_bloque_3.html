2. Restaurar un Curso Borrado

Archivo a modificar: c:\Users\cuent\Desktop\xd\IAScursos\src\main\java\com\crusos\application\usecase\AdminCourseUseCase.java

Ya que elegiste un Borrado Lógico (isActive=false), el curso sigue ahí. Podríamos hacer un endpoint para "Deshacer el borrado" y revivir el curso mágicamente.

// Añadir en AdminCourseUseCase.java:
public void restoreCourse(Long id) {
    Course course = courseRepositoryPort.findById(id)
        .orElseThrow(() -> new IllegalArgumentException("Curso no encontrado"));
        
    course.setIsActive(true); // ¡El curso revive y vuelve al catálogo público!
    courseRepositoryPort.save(course);
}
